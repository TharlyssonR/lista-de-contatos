# Lista de Contatos

Projeto simples de lista de contatos desenvolvido com HTML, CSS e JavaScript.

##  Funcionalidades

- Adicionar novos contatos
- Exibir lista de contatos
- Remover contatos
- Interface simples e fácil de usar

## 🛠 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

##  Aprendizados

Neste projeto pratiquei:

- Manipulação do DOM
- Eventos de clique (addEventListener)
- Criação de elementos com JavaScript
- Organização de código
- Lógica básica de programação

## Como usar

1. Digite o nome do contato
2. Clique no botão para adicionar
3. O contato aparecerá na lista
4. Para remover, utilize o botão de excluir (se tiver)
